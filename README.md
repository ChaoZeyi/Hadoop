## Hadoop

Hadoop是Apache开源组织的一个分布式计算开源框架，提供了一个分布式文件系统子项目(HDFS)和支持MapReduce分布式计算的软件架构。在Hadoop家族中，包含多个产品，像HBase、Hive、Zookeeper和Pig等，这里盗用网上的一张图（来自http://blog.fens.me/hadoop-family-roadmap/）：
